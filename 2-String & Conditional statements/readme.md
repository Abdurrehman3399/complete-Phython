#  Python Basics: Strings & Conditional Statements

##  Overview
This project covers important beginner-level Python concepts:
- Strings
- Conditional Statements

It helps learners understand how to work with text data and control the flow of programs.

##  Topics Covered
###  Strings
- What is a string?
- String creation and formatting
- String operations (concatenation, slicing, etc.)
- Common string methods

###  Conditional Statements
- `if` statement
- `if-else` statement
- `if-elif-else` ladder
- Nested conditions
- Logical operators (`and`, `or`, `not`)

##  Requirements
- Python 3.x  
- Jupyter Notebook (optional)

##  How to Run
1. Install Python  
2. Open the notebook file:
   `2_Strings_&_Conditional_Statements.ipynb`  
3. Run each cell step by step
