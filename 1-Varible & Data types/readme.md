# Python Basics: Variables & Data Types

## Overview
This project introduces the fundamental concepts of Python programming, focusing on:
- Variables
- Data Types

It is designed for beginners who are starting their journey in Python.

##  Topics Covered
- What is a variable?
- Rules for naming variables
- Different data types in Python:
  - Integer (`int`)
  - Float (`float`)
  - String (`str`)
  - Boolean (`bool`)
- Basic examples and usage

##  Requirements
- Python 3.x  
- Jupyter Notebook (optional, for running `.ipynb` file)

##  How to Run
1. Install Python on your system  
2. Open the notebook file:
   `1_Variables_&_Data_Types.ipynb`  
3. Run the cells step by step to understand the concepts  


##  Purpose
This project helps beginners understand the building blocks of Python, which are essential before moving to advanced topics.
