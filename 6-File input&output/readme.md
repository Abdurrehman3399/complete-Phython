#  Scientific Calculator in Python

##  Overview
This project is a **Scientific Calculator** built using Python.  
It performs both basic and advanced mathematical operations using functions and the `math` module.

It is designed for beginners to understand how to combine logic, functions, and libraries in Python.

---

##  Features
-  Addition  
-  Subtraction  
-  Multiplication  
-  Division  
-  Power (x^y)  
- √ Square Root  
-  Logarithm (log10)  
-  Trigonometric functions:
  - `sin`
  - `cos`
  - `tan`  
-  Continuous execution (loop-based)  
-  Error handling (invalid input, division by zero)

---

##  Requirements
- Python 3.x  
- No external libraries required (uses built-in `math` module)
