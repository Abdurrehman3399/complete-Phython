#  Python Basics: Dictionary & Set

##  Overview
This project introduces two important Python data structures:
- Dictionaries
- Sets

It is designed to help beginners understand how to store and manage collections of data efficiently.

##  Topics Covered

###  Dictionary
- What is a dictionary?
- Creating dictionaries
- Accessing values
- Adding and updating items
- Removing items
- Dictionary methods

###  Set
- What is a set?
- Creating sets
- Set operations (union, intersection, difference)
- Adding and removing elements
- Properties of sets (unique values, unordered)

## Requirements
- Python 3.x  
- Jupyter Notebook (optional)

##  How to Run
1. Install Python  
2. Open the notebook file:  
   `3_Dictionary_&_Set_in_Python_.ipynb`  
3. Run the cells step by step
